# NetPulse 项目结构说明

本文档描述 NetPulse 监控运维系统的目录与主要文件结构，便于论文插图、答辩展示或新人上手。**说明**：`target`、`node_modules`、`.git`、`.idea` 等构建与 IDE 目录已省略。

---

## 零、项目所用技术栈总览

以下为 NetPulse 中**实际使用**的技术与组件（以 `pom.xml`、`package.json`、`application.yml` 为准）。

### 后端

| 类别 | 技术/组件 | 版本或说明 | 用途 |
|------|-----------|------------|------|
| 语言与运行时 | Java | 21 | 后端开发语言 |
| 基础框架 | Spring Boot | 3.2.5 | 应用框架、自动配置、内嵌 Tomcat |
| Web | spring-boot-starter-web | - | REST 接口、MVC |
| 持久化 | Spring Data JPA | - | 实体与表、Repository |
| 数据库 | MySQL | 5.7 / 8.x（mysql-connector-j） | 业务数据（设备、告警、用户、审计等） |
| 数据库（可选） | H2 | runtime | 测试或本地可选 |
| 数据库迁移 | Flyway | 随 Spring Boot | 表结构版本化迁移（V2～V15） |
| 缓存/中间件 | Redis | spring-boot-starter-data-redis | 设备指标缓存、SNMP 结果缓存 |
| 消息队列（可选） | RabbitMQ | spring-boot-starter-amqp | 可选消息与异步 |
| 时序库 | InfluxDB | influxdb-client-java 6.11.0 | 设备状态时序、Telegraf 指标 |
| WebSocket | spring-boot-starter-websocket | - | Web 终端长连接 |
| 校验 | spring-boot-starter-validation | - | 请求参数校验 |
| 邮件 | spring-boot-starter-mail | - | 告警邮件通知 |
| SSH | JSch | 0.1.55 | Web 终端与 SSH 采集 |
| Telnet | Apache Commons Net | 3.10.0 | Web 终端与 Telnet 采集（端口 23） |
| SNMP | Snmp4j | 3.7.8 | 网络设备 SNMP 采集（v2c/v3） |
| 本机监控（可选） | OSHI | oshi-core 6.4.0 | 本机 CPU/内存/磁盘/网络 |
| 工具 | Lombok | - | 简化 getter/setter、Builder 等 |
| 测试 | spring-boot-starter-test | test | 单元/集成测试 |

### 外部依赖与部署

| 类别 | 技术/组件 | 说明 |
|------|-----------|------|
| 数据库 | MySQL | 主业务库，需单独部署 |
| 缓存 | Redis | 指标缓存，需单独部署（可选） |
| 时序库 | InfluxDB | 设备状态与 Telegraf 指标，可选 |
| 消息队列 | RabbitMQ | 配置中存在，可选 |
| 采集 Agent | Telegraf | 部署在 Linux 服务器上，将 CPU/内存等写入 InfluxDB |
| 反向代理（可选） | Nginx | 托管前端静态资源或反向代理后端 |
| 构建/运行 | Maven | 后端构建；Node.js / npm | 前端安装依赖与构建 |

### 简要归纳（论文可引用）

- **后端**：Java 21 + Spring Boot 3.2 + Spring Data JPA + MySQL + Redis +（可选）InfluxDB + Flyway + JSch（SSH）+ Commons Net（Telnet）+ Snmp4j（SNMP）+ Spring WebSocket + Spring Mail。
- **前端**：Vue 3 + Vue Router + Vite + Axios + xterm.js（终端）+ ECharts（图表）。
- **数据与中间件**：MySQL（业务）、Redis（缓存）、InfluxDB（时序）、（可选）RabbitMQ；Telegraf 用于 Linux 服务器指标采集。

## 二、后端 NetPulse（Spring Boot）

- **技术栈**：Java 21、Spring Boot 3.2、Spring Data JPA、Flyway、JSch（SSH）、Apache Commons Net（Telnet）、Snmp4j、InfluxDB 客户端、Redis。
- **包名**：`org.ops.netpulse`。
