# 论文题目与关键词

## 论文题目

**基于 Spring Boot 的运维监控平台设计与实现**

（注：学术写作中一般写 “Spring Boot” 两个单词，若学校要求中文可写“基于 Springboot 的运维监控平台设计与实现”。）

---

## 关键技术关键词（3～5 个）

从系统技术栈中选取 **3～5 个** 作为关键词，可直接用于论文“关键词”栏。

### 推荐组合（5 个）

**Spring Boot；运维监控平台；Vue；前后端分离；MySQL**

- **Spring Boot**：后端核心框架，与题目一致  
- **运维监控平台**：研究对象/主题  
- **Vue**：前端核心框架  
- **前后端分离**：架构特点  
- **MySQL**：主要业务数据存储  

### 备选组合（4 个）

**Spring Boot；运维监控平台；Vue；MySQL**

### 备选组合（3 个）

**Spring Boot；运维监控平台；Vue**
